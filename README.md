# Rick and Morty SPA
 
I created a Single Page Application with Javascript Vanilla.

In this project I used a RESTful API based on the serie of Rick and Morty: 
[Rick and Morty API](https://rickandmortyapi.com/)
 
To run the proyect:

npm install
npm run start
